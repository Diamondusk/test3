ระบบสรุปรายรับ-รายจ่าย (เชื่อมต่อ Google Sheet)

ดึงข้อมูลอัตโนมัติจาก Google Sheet ของคุณ กราฟอัปเดตทันทีเมื่อข้อมูลเปลี่ยนแปลง

วิธีเชื่อมต่อฐานข้อมูล Google Sheet ของคุณ:

  1. สร้างไฟล์ Google Sheet ขึ้นมาใหม่ และสร้างหัวคอลัมน์แถวแรกให้พิมพ์คำว่า: วันที่, ประเภท, หมวดหมู่, รายละเอียด, จำนวนเงิน (ห้ามพิมพ์ผิด)
  2. รูปแบบ "วันที่" แนะนำให้กรอกเป็น ปี-เดือน-วัน (เช่น 2024-01-25) เพื่อความแม่นยำ
  3. เมื่อกรอกข้อมูลเสร็จ ให้ไปที่เมนู File (ไฟล์) > Share (แชร์) > Publish to web (เผยแพร่ไปยังเว็บ)
  4. เลือกรูปแบบไฟล์จาก Web page เป็น Comma-separated values (.csv) แล้วกดปุ่ม Publish (เผยแพร่)
  5. คัดลอกลิงก์ที่เด้งขึ้นมา นำมาวางในช่องว่างด้านบน แล้วกดปุ่ม "โหลดข้อมูล" ได้เลยครับ!
